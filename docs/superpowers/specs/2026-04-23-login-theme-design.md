# 登录页面多主题设计方案

> 更新时间: 2026-04-23
>
> 目标: 为商城登录页面设计三套主题方案，支持主题切换和个性化内容展示

---

## 一、项目概述

### 1.1 背景
商城登录页面需要支持：
- 三套可切换的主题方案
- QQ 社交登录接入
- 个性化商品展示内容
- 响应式设计（PC端/移动端自适应）

### 1.2 三套主题方案

| 方案 | 名称 | 定位 | 目标用户 |
|------|------|------|----------|
| Scheme 1 | 兴趣电商 | 活力、多彩、吸引人 | 年轻用户、潮流爱好者 |
| Scheme 2 | 大厂稳重 | 简洁、可信度高、功能多样 | 成熟用户、商务人士 |
| Scheme 3 | 轻奢简约 | 高端、简约、精致 | 高消费群体、品质追求者 |

---

## 二、Scheme 1: 兴趣电商（Youth/Vibrant）

### 2.1 视觉风格

**配色方案:**
- 主色: `#FF6B35` (活力橙)

- 强调色: `#00D9C0` (青绿色), `#FF4D94` (粉色)

## 五、主题切换机制

### 5.1 切换组件设计

```tsx
// ThemeSwitcher 组件位置: 页面右上角或右下角浮动按钮
// 样式: 三个主题色圆点 + 名称

// 主题状态存储: localStorage.setItem('login-theme', 'vibrant' | 'stable' | 'luxury')
```

### 5.2 主题数据结构

```typescript
type LoginTheme = 'vibrant' | 'stable' | 'luxury'

interface ThemeConfig {
  id: LoginTheme
  name: string
  primaryColor: string
  backgroundGradient: string
  cardStyle: 'rounded' | 'flat' | 'minimal'
  socialStyle: 'colorful' | 'subtle' | 'outline'
}
```

## 七、个性化商品展示

### 7.1 商品展示位置

| 主题 | 展示位置 | 商品数量 |
|------|----------|----------|
| Vibrant | 左侧品牌区，轮播 | 4-6 个 |
| Stable | 右侧功能区，网格 | 3-4 个 |
| Luxury | 全屏背景，单品大图 | 1 个 |

### 7.2 商品数据接口

```typescript
// 获取登录页展示商品
GET /api/v1/rec/login-showcase?theme=vibrant&limit=6

// 响应
{
  items: [
    {
      sku_id: string
      title: string
      price: number
      original_price: number
      image: string
      discount?: number  // 如: 20 表示 8折
    }
  ]
}
```

### 7.3 交互行为

- 点击商品: 跳转商品详情页 (无需登录可浏览)
- 登录提示: 商品卡片底部显示 "登录后享受更多优惠"
- 跳过行为: 整个商品区可点击跳过登录
