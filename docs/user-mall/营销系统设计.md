```
用户行为 → 触发营销规则 → 优惠计算 → 订单结算
    │
    ├── 注册 → 新人礼包（优惠券+积分）

    ├── 浏览 → 浏览记录 → 推荐优化

    ├── 支付 → 积分返还 → 会员成长
    └── 评价 → 评价奖励（积分/优惠券）
```

## 2. 优惠券系统

### 2.1 优惠券类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `fixed` | 固定金额减免 | 满100减20 |
| `percent` | 百分比折扣 | 满200享9折 |
| `shipping` | 免运费券 | 指定商品免运 |
| `gift` | 礼品券 | 新用户送礼 |

### 2.2 优惠券规则

| 规则项 | 说明 |
|--------|------|
| 发放方式 | 管理员发放/用户领取/活动奖励/邀请得券 |
| 使用门槛 | 最低消费金额 |
| 适用范围 | 全平台/指定分类/指定商品 |
| 有效期 | 固定日期 / 领取后N天有效 |
| 使用限制 | 每人限领数量 / 整个活动限发数量 |
| 叠加规则 | 是否可与其他优惠券叠加 |

### 2.3 优惠券叠加规则

| 场景 | 叠加规则 |
|------|---------|
| 店铺券 + 平台券 | 可叠加（但各有门槛） |
| 商品折扣 + 优惠券 | 优惠券在折扣后计算 |
| 秒杀商品 | 不与其他优惠叠加 |
| 积分抵扣 | 可与优惠券叠加 |

### 2.5 优惠计算逻辑

```go
// CalculateDiscount 计算优惠
func CalculateDiscount(cart *Cart, coupon *UserCoupon, user *User) *Discount {
    result := &Discount{}

    // 1. 获取商品原价
    goodsAmount := cart.GetGoodsAmount()

    // 2. 检查门槛
    if goodsAmount < coupon.MinAmount {
        result.CanUse = false
        result.Reason = fmt.Sprintf("未达最低消费 %d 元", coupon.MinAmount/100)
        return result
    }

    // 3. 检查适用范围
    if !coupon.IsApplicableTo(cart.Items) {
        result.CanUse = false
        result.Reason = "优惠券不适用于当前商品"
        return result
    }

    // 4. 计算优惠
    switch coupon.Type {
    case "fixed":
        result.Discount = coupon.Value
    case "percent":
        result.Discount = goodsAmount * coupon.Value / 100
        if coupon.MaxDiscount > 0 && result.Discount > coupon.MaxDiscount {
            result.Discount = coupon.MaxDiscount
        }
    }

    // 5. 积分抵扣（独立计算）
    pointsDiscount := CalculatePointsDiscount(user.Points, goodsAmount-result.Discount)

    result.CanUse = true
    result.CouponDiscount = result.Discount
    result.PointsDiscount = pointsDiscount
    result.TotalDiscount = result.Discount + pointsDiscount
    result.PayAmount = goodsAmount - result.TotalDiscount

    return result
}
```

## 3. 秒杀系统

### 3.1 秒杀场次

| 时段 | 名称 | 持续时间 |
|------|------|---------|
| 10:00-12:00 | 早场 | 2小时 |
| 14:00-16:00 | 下午场 | 2小时 |
| 20:00-22:00 | 晚场 | 2小时 |
| 00:00-06:00 | 午夜场 | 6小时 |

### 3.2 秒杀规则

| 规则项 | 说明 |
|--------|------|
| 库存锁定 | 下单后锁定库存，支付后确认 |
| 超卖防护 | Redis 原子扣减，防止超卖 |
| 限购数量 | 每人每场限抢N件 |
| 活动预热 | 开抢前X分钟显示倒计时 |
| 抢购结果 | 抢购成功/失败/排队中 |

### 3.3 秒杀流程

```
1. 活动预热
   ├── 用户进入秒杀页
   ├── 显示即将开始的场次和商品
   └── 可设置抢购提醒

2. 抢购开始
   ├── 倒计时结束
   ├── 前端开始请求（带前后端时间校准）
   └── 后端接收请求

3. 库存校验（Redis）
   ├── Lua 脚本原子扣减
   ├── DECR 并发安全
   └── 库存不足直接返回

4. 限购校验
   ├── 检查用户已购数量
   ├── 超过限购返回失败

5. 下单锁定
   ├── 创建订单（状态待支付）
   ├── 冻结库存（reserved）
   └── 返回抢购结果

6. 支付确认
   ├── 支付成功
   ├── 确认冻结库存
   └── 抢购完成
   │
   └── 超时未支付
       ├── 释放冻结库存
       └── 恢复 Redis 库存
```

### 3.5 Redis 秒杀实现

```go
// Redis 秒杀库存 Key
const seckillStockKey = "seckill:stock:%s:%s" // seckill:stock:{activity_id}

// Lua 脚本：原子扣减库存
const seckillScript = `
local stock = redis.call('GET', KEYS[1])
if not stock then
    return -1  -- 活动不存在
end
stock = tonumber(stock)
if stock <= 0 then
    return 0  -- 库存不足
end
redis.call('DECR', KEYS[1])
return 1  -- 扣减成功
`

func Seckill(ctx context.Context, activityID, userID string) (bool, error) {
    // 1. 检查用户购买限制
    boughtKey := fmt.Sprintf("seckill:bought:%s:%s", activityID, userID)
    bought, _ := rdb.Get(ctx, boughtKey).Int()
    if bought >= limitPerUser {
        return false, errors.New("已达购买上限")
    }

    // 2. 原子扣减库存
    stockKey := fmt.Sprintf(seckillStockKey, activityID)
    result, err := rdb.Eval(ctx, seckillScript, []string{stockKey}).Int()
    if err != nil {
        return false, err
    }
    if result <= 0 {
        return false, errors.New("库存不足")
    }

    // 3. 记录购买数量
    rdb.Incr(ctx, boughtKey)
    rdb.Expire(ctx, boughtKey, 24*time.Hour)

    return true, nil
}
```

### 6.2 拼团流程

```
1. 发起拼团
   ├── 选择商品
   ├── 选择拼团人数
   ├── 创建拼团订单（待支付）
   └── 生成拼团链接

2. 分享邀请
   ├── 分享拼团链接
   └── 好友点击参团

3. 参团
   ├── 查看拼团详情
   ├── 支付下单
   └── 加入拼团

4. 拼团成功
   ├── 达到成团人数
   ├── 所有订单确认
   └── 商家发货

5. 拼团失败
   ├── 超时未成团（24小时）
   ├── 全额退款
   └── 释放库存
```

## 10. 优惠叠加计算

### 10.1 叠加优先级

```
商品总价
    ↓
商品折扣（秒杀/促销/会员价）
    ↓
优惠券（店铺券/平台券）
    ↓
积分抵扣
    ↓
应付金额
```

### 10.2 计算公式

```
商品总价 = Σ(商品单价 × 数量)

折扣后金额 = 商品总价 × 折扣率

优惠券后金额 = 折扣后金额 - 优惠券金额
  （若优惠券为百分比，则 = 折扣后金额 × (1 - 优惠券%）

积分抵扣后金额 = 优惠券后金额 - min(积分可抵扣金额, 优惠券后金额×50%)

最终金额 = max(积分抵扣后金额, 0.01)  // 最低0.01元
```

### 10.3 计算示例

```
商品原价：1000元
商品折扣：9折 → 900元
会员折扣：95折 → 855元
优惠券：满500减100 → 755元
积分：1000积分（抵扣10元）→ 745元

最终支付：745元
```

// 订单服务调用营销服务
type MarketingService interface {
    CalculateDiscount(ctx context.Context, req *DiscountRequest) (*DiscountResponse, error)
    UseCoupon(ctx context.Context, userID, couponID, orderNo string) error
    RewardPoints(ctx context.Context, userID, orderNo string, amount int) error
    UpdateMemberGrowth(ctx context.Context, userID, orderNo string, amount int) error
}

// 创建订单时
func (s *OrderService) CreateOrder(req *CreateOrderRequest) (*Order, error) {
    // 1. 计算优惠
    discount, err := s.marketing.CalculateDiscount(ctx, &DiscountRequest{
        UserID: req.UserID,
        Items:  req.Items,
        CouponID: req.CouponID,
    })

    // 2. 创建订单
    order := &Order{
        TotalAmount: req.Items.GetAmount(),
        DiscountAmount: discount.TotalDiscount,
        PayAmount: discount.PayAmount,
    }

    // 3. 使用优惠券
    if req.CouponID != "" {
        await s.marketing.UseCoupon(ctx, req.UserID, req.CouponID, order.OrderNo)
    }

    return order, nil
}

// 订单完成后
func (s *OrderService) OnOrderPaid(order *Order) {
    // 1. 奖励积分
    go s.marketing.RewardPoints(order.UserID, order.OrderNo, order.PayAmount)

    // 2. 更新会员成长值
    go s.marketing.UpdateMemberGrowth(order.UserID, order.OrderNo, order.PayAmount)

    // 3. 发送通知
    go s.notification.SendOrderPaid(order)
}

```typescript
// 购物车页面需要展示优惠信息
const CartPage = () => {
  const { items, selectedIds } = useCartStore()
  const { coupons, availableCoupons } = useAvailableCoupons(selectedIds)
  const { points, canUsePoints } = useUserPoints()

  const calculateTotal = () => {
    const goodsAmount = items
      .filter(i => selectedIds.includes(i.id))
      .reduce((sum, i) => sum + i.price * i.quantity, 0)

    // 选中可用优惠券中最大优惠
    const bestCoupon = coupons.reduce((best, c) =>
      c.discount > best.discount ? c : best, { discount: 0 })

    const afterCoupon = goodsAmount - bestCoupon.discount
    const pointsDiscount = canUsePoints
      ? Math.min(afterCoupon * 0.5, points / 100 * 100)
      : 0

    return {
      goodsAmount,
      couponDiscount: bestCoupon.discount,
      pointsDiscount,
      total: afterCoupon - pointsDiscount
    }
  }

  return (
    <div>
      {/* 商品列表 */}
      <CartList items={items} />

      {/* 优惠券选择 */}
      <CouponSelector coupons={availableCoupons} />

      {/* 积分抵扣 */}
      <PointsSelector canUse={canUsePoints} points={points} />

      {/* 价格汇总 */}
      <PriceSummary {...calculateTotal()} />
    </div>
  )
}
```
